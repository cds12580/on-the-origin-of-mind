On the Origin of Mind

A Matter of Senses, and Time

Glossary

Mink space: The expanded Minkowski space; comprising all sensory inputs in the umwelt of the organism, plus time.

Time horizon: The organism’s vertical limits of FOC

Mink space topology: A set of adjacent world points

Event pattern: An isolated and compaginated set of adjacent world points in mink space (point cloud). Event patterns are semantic (explicit) memory

Snatching: Observing an event pattern from its umwelt through sensory organs

Matching: Assigning the most relevant event stack to any snatched event pattern through jumping or rolling

Fetching: Obtaining the closest match from a selected event stack

Patching: Complementing the umwelt (present plane) with a fetched event pattern

Hatching: One of two methods of pattern creation; refers to isolating an event pattern from its environment for the first time. Primary hatching concerns a first for the super-organism; secondary hatching concern a first for the individual organism

Latching: One of two methods of pattern creation; refers to re-shaping or re-arranging event patterns into a new entity. Primary latching concerns a first for the super-organism; secondary hatching concern a first for the individual organism

Compagination: The process of joining two or more concurrent or successive event patterns (pairing)

Rolling: One of two concatenation methods linking event patterns in a chain of thought; we compared it to a hallway connecting adjacent rooms in a building. The user is aware of the in-between

Jumping: One of two concatenation methods linking event patterns in a chain of thought; we compared it to an elevator connecting two floor in a building through similarly shaped openings. The user is unaware of the in-between

Cloud band, also: world-point cloud: An agglomeration of adjacent world points comprising multiple event patterns. A cloud band is episodic (explicit) memory

Event stack: A theoretical construct representing all alike-shaped event patterns in chronological order

Stack bleed: snatched event patterns include always parts of its environment; these world points provide context and connotation; the event bleed can contain TLPs and LLPs

Mold, also: event envelope: The flattened and weighted event stack; a schema or stereotype for a certain event

Top-level event pattern (TLP): Such event patterns that form part of the ROC and enter focus

Lower-level event pattern (LLP): Fill patterns that make up large part of the umwelt landscape but are observed only sub-consciously

In-orgs: World points inside the organism

Ex-orgs: World points outside of the organism

Vertical: Along the time axis

Horizontal: In the same present plane

Triangular thinking: A thought that originates and terminates in the here-now

Chain of thought: A series of overlapping event patterns joined by rolling or jumping

Habit or routine: A pre-timed chain of thought that has been stripped of unnecessary detours and event bleed in mink space (pruned). A habit runs unattended in the sub-conscious. Habits are procedural (implicit) memory

Field of consciousness (FOC): The conceivable umwelt limits for the organism

Realm of consciousness (ROC): The set of hatched event patterns for the organism

Field of thought (FOT): The actively for thinking and feeling used portion of FOC

Constellation: Can refer to the relative position of event points in the event mold or the corresponding mapping to the physical brain

Umwelt (German: environment): The umwelt concept according to Jakob von Uexküll

Weltbild (German: world view): The inner representation of the organism’s umwelt; the largest piece of continuous and consistent mink-space fabric for the organism
